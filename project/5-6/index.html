Research and development on building the integrative support environment for modeling and simulation

Yasuo Fujii

DSP technology Co.Ltd

e-mail: yasuo.f@dsptec.co.jp

The goal of our study is to develop an integrative support environment for Mathematical analysis and computer simulation study in neuroinformatics research, which is a basic platform for researchers engaged in neuroinformatics in vision project and else.

Last year, we have been ported the SATELLITE in MS-Windows. And we confirmed that it can run with the UNIX environment by a little modification. It is because this code was made not to depend on the platforms.

This year, in order to develop a function of real-time simulation on the SATELLITE we first construct the fundametal systems such as code generation, parameter trunning, monitoring, data loging. Next we integrate them in the SATELLITE environment. The fusion of a mathematical model made by SATELLITE and the physiological system will make us utilize model based simulation study.
