Development of computer support environment for neuroinformatics

Keisuke Takebe

Computer Center, Nagaoka National College of Technology

e-mail: takebe@nagaoka-ct.ac.jp

In order to reveal the characteristics of biological neural systems, a suitable simulation and analysis environment is very important. The basic concept of system analysis is in the form of the cycle: data analysis, modeling, computer simulation, evaluation and experimental testing.

The goal of this study is to develop "Neuroinformatics Desktop Environment" (NI-DTE) which is a computer support desktop environment package for neuroinformatics.

First and second year, we have developed a software named ``Neuroinformatics SATELLITE'' (NI-SATELLITE). It is brush up version of the system analysis total environment " SATELLITE" (developed by the biological and physiological engineering laboratory, TUT.)
